Versión 1.0.0 - 2025
Descripción del Sistema

El Sistema Comercial Integral Argentina es una aplicación de gestión empresarial desarrollada en Python 3.13.9 con Django y base de datos MySQL. Está diseñado para pequeñas y medianas empresas que necesitan controlar stock, realizar ventas y compras, emitir facturas electrónicas a AFIP, manejar múltiples listas de precios, gestionar clientes, proveedores y sucursales.

Detalles Técnicos
  • Lenguaje: Python 3.13.9
  • Framework: Django 5.x
  • Base de Datos: MySQL 8.0
  • Interfaz: HTML + Bootstrap 5 + CSS
  • Integraciones: AFIP, Impresora Fiscal, Email SMTP
  • Seguridad: Roles, permisos y auditorías de acceso
