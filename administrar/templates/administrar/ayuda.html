Centro de Ayuda del Sistema

Guía rápida para usuarios principiantes

Bienvenido al Sistema Comercial CARSOFT. Este sistema fue desarrollado para facilitar la gestión de ventas, compras, stock, clientes, proveedores y facturación (incluyendo conexión con AFIP).

Está pensado para pequeñas y medianas empresas de Argentina, y cuenta con una interfaz simple e intuitiva.

Para ingresar, escriba su nombre de usuario y contraseña asignados por el administrador. Si olvida su contraseña, comuníquese con el administrador del sistema.

El menú principal muestra las diferentes secciones del sistema. Cada tarjeta tiene un color e ícono que representan su función (Ventas, Compras, Productos, Clientes, etc.).

  • Ventas: registrar facturas y comprobantes.
  • Productos: gestionar stock y precios.
  • Clientes: administrar datos y cuentas corrientes.
  • Proveedores: control de compras y cuentas a pagar.
  • Configuración: ajustar datos de empresa, AFIP, correo, etc.

  1. Seleccione el cliente.
  2. El sistema cargará automáticamente el tipo de factura según su condición fiscal.
  3. Agregue los productos desde la lista o código de barras.
  4. Seleccione la lista de precios (efectivo, tarjeta o cuenta corriente).
  5. Presione Confirmar Venta.
  6. El sistema generará la factura y podrá imprimirla o enviarla por correo electrónico.

En esta sección puede dar de alta nuevos productos, modificar precios o ajustar stock.

  • Puede definir hasta 4 listas de precios.
  • El stock se actualiza automáticamente al realizar ventas o compras.
  • Se pueden exportar listas de productos a Excel o PDF.

Permite registrar clientes con todos sus datos fiscales, condición frente al IVA y límite de crédito.

Si un cliente supera su límite, el sistema mostrará una alerta, aunque el administrador puede autorizar la venta manualmente.

El sistema puede emitir comprobantes electrónicos (Factura A, B o C) conectándose con los servicios de AFIP mediante certificados digitales.

También se puede trabajar con impresora fiscal o con facturación tradicional.
